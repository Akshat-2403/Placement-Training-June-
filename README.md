# 📘 Placement Training – June

Welcome to the **Placement Training Repository**!  
This repo contains curated resources, notes, and practice materials to help students prepare for campus placements. It covers **aptitude, reasoning, programming, and interview readiness** in a structured way.

---

## 🚀 Objectives
- Strengthen **core aptitude** (quantitative, logical, verbal).
- Build confidence in **coding interviews**.
- Practice **group discussions & HR rounds**.
- Provide **visual progress tracking** of covered topics.

---

## 📂 Repository Structure
- `Aptitude/` → Quantitative aptitude problems (Arithmetic, Algebra, Probability, etc.)
- `Reasoning/` → Logical reasoning, puzzles, seating arrangements.
- `Verbal/` → Grammar, comprehension, vocabulary practice.
- `Programming/` → Data structures, algorithms, coding challenges.
- `Interview/` → HR questions, behavioral tips, mock interview scripts.
